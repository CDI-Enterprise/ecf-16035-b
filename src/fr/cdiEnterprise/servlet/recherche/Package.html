Package fr.cdiEnterprise.servlet.recherche

Ce package contient toutes les Servlets qui gèrent les actions sur la page de recherches d'entreprise

Liste des Servlets:

RechControl : réoriente le flux vers les Servlets suivantes selon l'URL associée

RechAffichageNew : Gère l'affichage de la page

RechListe : Gère la génération d'une liste d'entrprises selon une recherche

RechEnregistrer : Gère l'enregistrement d'une recherche favorite

RechFavAfficher : Gère l'affichage d'une recherche favorite

RechFavSuppr : Gère la suppression d'une recherche favorite
